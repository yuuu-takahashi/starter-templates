# shared/tsconfig/

各テンプレートの `tsconfig.json` の共通ソースです。`yarn generate:configs` で以下にコピーされます。

- **react/** — `templates/react/tsconfig.json`, `tsconfig.app.json`, `tsconfig.node.json`
- **nextjs/** — `templates/nextjs/tsconfig.json`
- **rails/** — `templates/rails/tsconfig.json`

編集後は `yarn generate:configs` を実行して各テンプレートへ反映してください。
