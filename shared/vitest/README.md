# shared/vitest/

Next テンプレート用の Vitest 設定の共通ソースです。`yarn generate:configs` で以下を生成します。

- **vitest.config.vite-react.ts** → `templates/nextjs/vitest.config.ts`
- **vitest.setup.ts** → `templates/nextjs/vitest.setup.ts`
- **vitest-env.d.ts** → `templates/nextjs/vitest-env.d.ts`

編集後は `yarn generate:configs` を実行して反映してください。
